Create a new command to list Fah options filtered by given verbosity level.
Parameters:
Name Type Argument Default Description
level string <optional>
 explicit The verbosity level, determines which options are shown. This can be explicit (options set by user), implicit (explicit + options with default values), unset (implicit + unset options), or all. Any other value will default to explicit.
slot number <optional>
 The number of the slot to be configured
